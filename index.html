1. Latar Belakang Shopee

Shopee merupakan platform e-commerce terdepan di Asia Tenggara yang menghubungkan jutaan pembeli dan penjual. Dengan volume transaksi yang masif setiap detiknya, Shopee menghadapi tantangan manajemen data yang sangat kompleks, mulai dari inventaris barang, manajemen keranjang, hingga proses autentikasi pengguna. Tanpa sistem yang terstruktur, pengembangan fitur baru menjadi sangat berisiko (rawan bug) dan sulit dipelihara. Oleh karena itu, penerapan paradigma Object-Oriented Programming (OOP) menjadi kebutuhan mutlak agar arsitektur sistem tetap skalabel dan efisien.

2. Mengapa Masalah Ini Lebih Baik Diselesaikan dengan OOP?

Pendekatan OOP menawarkan solusi atas kompleksitas sistem e-commerce melalui:

  • Encapsulation: Menyembunyikan detail sensitif (seperti saldo atau informasi pembayaran) di dalam objek untuk menjaga keamanan.
  • Inheritance: Memungkinkan pembuatan kategori produk (elektronik, fashion, dsb) secara hierarkis, yang efisien dalam penulisan kode.
  • Abstraction: Menyederhanakan proses kompleks (seperti kalkulasi pajak atau checkout) menjadi method yang mudah dipanggil.
  • Modularity: Memungkinkan tim pengembang bekerja pada bagian sistem yang berbeda tanpa saling merusak fungsi satu sama lain.

3. Langkah-Langkah Menyelesaikan Masalah

  1. Analisis: Mengidentifikasi objek utama (User, Product, Cart, Transaction).
  2. Desain Kelas: Menentukan atribut dan method berdasarkan prinsip OOP.
  3. Pemecahan Modul: Membagi fungsi ke dalam 5 file kode terpisah untuk menjaga kebersihan sistem (Clean Code).
  4. Implementasi & Relasi: Menghubungkan antar objek (misal: Cart berisi daftar Produk).
  5. Pengujian: Memastikan sistem berjalan sesuai alur transaksi e-commerce.

Source Code 1: product.dart (Abstraksi)

abstract class Product {
  String name;
  double price;
  Product(this.name, this.price);
  void displayInfo();
}

Source Code 2: electronic.dart (Inheritance)

import 'product.dart';
class Electronic extends Product {
  Electronic(String name, double price) : super(name, price);
  @override
  void displayInfo() => print("Elektronik: $name - Rp$price");
}

Source Code 3: cart.dart (Composition)

import 'product.dart';
class Cart {
  List<Product> items = [];
  void addItem(Product p) => items.add(p);
  double getTotal() => items.fold(0, (s, i) => s + i.price);
}

Source Code 4: user.dart (Encapsulation)

import 'cart.dart';
class User {
  String _name;
  Cart cart = Cart();
  User(this._name);
  String get name => _name;
}

Source Code 5: main.dart (Eksekusi)

import 'user.dart';
import 'electronic.dart';

void main() {
  var user = User("Serlin & Jeinnet");
  user.cart.addItem(Electronic("Smartphone", 5000000));
  print("Total Pembayaran: Rp${user.cart.getTotal()}");
